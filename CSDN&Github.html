CSDN是国内最大的技术社区之一，里面有大量的开发资源。并且还有非常多PC日常使用过程中遇到的问题的解决办法。
但是注意，里面有一些坏人。把免费开源的资源和项目收费。使用的时候一定要注意。
Github，全球最大的开源代码托管平台。功能和资源强大到难以置信。这个网页的部署就是依托github pages静态网页托管功能。
Github的具体使用技巧相当多，甚至你可以搜到数十个小时的教学视频。在这里也不具体说了，主要介绍一下如何在国内快速稳定的登录Github。(不使用加速器和梯子)
首先使用win+r调出运行窗口，输入cmd打开命令提示符窗口。
将以下内容复制上去"C:\Program Files (x86)\Microsoft\Edge\Application\msedge.exe" --host-rules="MAP github.com octocaptcha.com, MAP github.githubassets.com yelp.com, MAP *.githubusercontent.com githubusercontent.com" --host-resolver-rules="MAP octocaptcha.com 20.27.177.113, MAP yelp.com 199.232.240.116, MAP githubusercontent.com 199.232.176.133"（具体的path要按照你电脑上的文件路径来写，此处给出的应该是win 11的默认路径。复制时引号也要带上。）后按下回车键，会发现打开了一个浏览器。
上述过程中，用指定路径启动Edge浏览器，通过参数将GitHub相关域名映射到其他域名及IP，改变了访问指向。
另外，Github的使用也需要你注册自己的账户。
